二、环境监控

2.1 模块业务简述

该页面实时采集设备数据综合展示，包括系统内创建对象数据统计，以及环境检测设备上传数据。根据养殖分组关联的设备回传的指标数据进行展示。

2.2 业务说明

元素组件说明取值逻辑 / 特别变更
按钮【日期筛选】 用户自填入日期（yy-mm-dd），根据日期显示对应时间的下方所有信息卡片。 —
按钮【导出数据】 点击导出，弹出日期筛选弹窗，选择性下载。默认取每个当天的 0–24 点的设备数据。 —
信息卡片【分组总数】 统计养殖分组数量 取值：养殖管理 → 养殖分组
信息卡片【设备总数】 统计设备总数量 取值：设备管理
信息卡片【指标数量】 统计数据指标总数量 取值：设备管理 → 数据指标管理
信息卡片【异常报警】 统计当天异常报警数量 取值：告警列表（当天数量）
【环境指标数据】 左侧为养殖分组列表，右侧五个标签页显示不同分类环境数据。 变更【数据趋势】由原来的独立按钮变更到右侧指标分类的第五个 Tab。

三、设备管理

3.1 模块业务简述

由原本的【产品分类】、【产品管理】、【设备管理】多个页面整合成一个页面视图，主显示设备信息，【产品分类】、【产品管理】换成按钮形式，在页面左侧树组件中显示，有独立弹窗页面进行管理。

💡 新型设备创建模拟路径
  1. 创建产品分类【水质仪】
  2. 创建产品【多参数水质仪】，型号【XH-600A】；关联分类【水质仪】，选择该产品关联哪些【数据指标】（水温、pH 值等）
  3. 新增设备【多参数水质仪 #01】，选择所属产品，输入必要信息项，完成创建

3.2 产品数据指标管理 — 业务说明

页面位置：作为按钮放置在设备列表【新增设备】左侧。

根据对接的设备，管理所有设备回传数据类型，不开放给用户增删，只做阈值设置。在【产品】创建时，可以选择关联哪些数据项；在设备详情中也需要展示对应关联了哪些数据项。

4.2 标准计划（原养殖计划）— 业务说明

创建养殖模版，设定标准计划的周期节点的投喂数据和预计增重数据，用于【养殖项目】创建时，插入标准计划，对比实际项目进行中的投喂数据和增重数据，用于数据对照参考。

📌 具体细节

参考原型中的标准计划模版详情抽屉，包含：计划名称、适用品种、养殖周期、各阶段投喂策略、预计增重曲线等。

4.3 分组管理（原养殖分组）— 业务说明

分组管理创建，需要创建网箱，每个网箱需要关联投喂哪个投喂设备的出口（投喂设备 → 管道 → 出口）。

分组也需要关联环境检测设备。这样查看环境监控页面时，可按分组查看对应关联设备的数据。

🔗 数据关联
  • 分组 → 网箱（一对多）
  • 网箱 → 投喂设备出口（多对一）
  • 分组 → 环境检测设备（一对多）
  • 分组 → 标准计划模版（多对一，可选）

4.4 投喂计划（原投喂管理）— 业务说明

创建计划，设定每日固定的投喂时间点和投喂量。然后将计划绑定投喂设备出口（投喂设备 → 管道 → 出口）。

原型页面上显示有网箱，如果在前面分组管理中，网箱绑定了该出口，则显示该出口对应的网箱，方便操作人员确认信息。没有绑定则不显示。

4.5 投喂详情（原投喂管理）— 业务说明

该页面左侧显示所有投喂设备的树状结构（投喂设备 → 管道），右侧为选择管道后，显示相关管道的信息和所有出口对应的信息。

⚠️ 开发注意

根据设备厂商 AKVA 接口能力开发，有不合理项酌情删减。

4.6 养殖日志 — 业务说明

无太大改动，具体参考原型。包含 7 个 Tab：投喂记录、养殖打样、精准统计、鱼损记录、料仓库存、鱼苗采购、操作管理。

4.7 水下管理（原视频监控）— 业务说明

去除监控设备单独页面管理，监控设备的创建连接，放回【设备管理模块】，根据后端对接所需信息，定义创建所需填入信息。

该模块仅显示已添加的视频流展示。

五、养殖项目（原项目管理）

5.1 业务说明

养殖项目模块从宏观维度管理养殖项目的组织与执行。项目包含多个子项目，每个子项目通过「养殖分组与计划对照」将分组、网箱和养殖计划关联起来，实现项目级别的养殖任务编排和进度跟踪。

功能说明
项目管理创建养殖项目，包含项目名称、编号、区域、负责人、描述等信息。每个项目可包含多个子项目。
子项目与计划对照子项目的核心功能是将具体的养殖分组（及其网箱）与养殖计划模版关联，形成执行映射。
执行跟踪子项目详情中展示投喂趋势图和重量趋势图，对比计划目标值与实际执行值的偏差。

5.2 项目创建顺序关系说明

  1. 创建项目：创建一个养殖项目，填写必要的项目信息，保存完，创建的项目出现在项目列表视图。
  2. 创建子项目：点击刚创建的【项目】，在详情中，创建【子项目】，除了必要的项目信息，需要为其配置所作用的分组，以及选择该分组下的某几个网箱，然后给它关联某一个【标准计划】模版进行项目数据对照。
  3. 子项目详情：创建完【子项目】后，在【总项目】详情中，可看到【子项目】及【子项目所关联的分组信息】，点击子项目详情，跳转展示子项目信息。具体展示信息未有大的改动，具体细节和样式参照原型。
  4. 子项目 - 养殖分组详情：在【子项目详情】中，可查看到所关联的【养殖分组】的信息，点击进入单个【养殖分组】的详情。可见【周期配置与执行】、【投喂记录】、【抽样记录】三个 Tab：
    • 【周期配置与执行】：该分组各个网箱的实际数据与创建时关联的【标准计划】的实验数据对比。
    • 【投喂记录】：该分组下被选择的网箱，的所有投喂记录，可人工提交投喂记录。
    • 【抽样记录】：手动提交的抽样记录。
